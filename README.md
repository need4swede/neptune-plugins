## Plugin Installation

In Jellyfin, open **Dashboard → Plugins → Repositories** and add:

```
https://raw.githubusercontent.com/need4swede/neptune-jellyfin-plugin/main/manifest.json
```

Open the **Catalog** tab, install **Neptune**, and restart the server.
